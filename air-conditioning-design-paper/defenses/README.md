# 答辩材料包

本目录用于存放课程设计论文的答辩材料。

建议使用顺序如下：

1. 先看 `01-答辩PPT内容提纲.md`
   用它快速整理答辩 PPT 的页序、每页核心信息和建议展示重点。
2. 再看 `02-答辩逐字稿-8到10分钟.md`
   这是可以直接练习朗读的主讲稿。
3. 然后熟悉 `03-答辩问答准备.md`
   这里整理了老师较可能追问的问题与回答思路。
4. 最后带上 `04-答辩速记卡.md`
   适合答辩前几分钟快速复习关键数字和结论。

本材料包的默认场景：

- 课程设计答辩
- 时长约 `8` 到 `10` 分钟
- 以“研究背景 -> 方法 -> 结果 -> 结论与局限”作为主线
- 以当前五城、15 组工况的仿真结果为依据

当前建议你重点记住的表达原则：

- 不把 `VRF+DOAS` 和 `FCU+DOAS` 预设为谁一定更优。
- 强调统一母版建筑和统一边界，这是比较公平性的基础。
- 强调 `Ideal Loads` 的作用是隔离气候驱动负荷，而不是替代实际系统。
- 强调当前 `FCU+DOAS` 仅统计了空调电耗，尚未纳入锅炉燃料，因此不能直接下“综合能耗更优”的最终结论。
